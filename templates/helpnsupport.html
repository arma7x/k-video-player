# Scanning files on high capacity storage take a long time to finish. Please 'Kill App' if you think the app was hang
# The 'video' and 'subtitle'(optional) files must be in the same folder. Both files must use the same name
# Supported subtitles are VTT and SRT
# Press Call button 3 times in a row to force-kill the app
# Empty search keyword will reset search result
# Shortcut key
  • Arrow Up or Arrow Down to adjust volume
  • Arrow Left or Arrow Right to rewind/fast-forward video duration by 10 second
  • Double-Click Arrow Left or Arrow Right to rewind/fast-forward video duration by 30 second
  • Key 1 to decrease playback rate by 0.1x
  • Key 2 to reset playback rate
  • Key 3 to increase playback rate by 0.1x
  • Key 4 to decrease subtitle height by 1%
  • Key 6 to increase subtitle height by 1%
  • Key 7 to decrease subtitle font size by 1%
  • Key 9 to increase subtitle font size by 1%
  • Key * to decrease contrast by 1%
  • Key # to increase contrast by 1%
  • Key 8 to decrease brightness by 1%
  • Key 0 to increase brightness by 1%
  • Call to take screenshot
